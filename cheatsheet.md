### If Statements

```C#
if (number > 8 || number < -3)
{
  Console.WriteLine("Good");
}
```

```python
if (number > 8 or number < -3):
  print("Good")
```

### If-else Statements

```C#
if (score < 70 && score >= 0)
{
  Console.WriteLine("Work a little harder");
}
else
{
  Console.WriteLine("Good job!");
}
```

```python
if (score < 70 and score >= 0):
  print("Work a little harder")
else:
  print("Good job!")
```
